DOM é a sigla para Document Object Model

Conjunto de objetos dentro do navegador que vai dar acesso aos componentes internos do site. O DOM não funciona dentro do Nodejs. O DOM está presente quando rodamos o javascript dentro do navegador.

Árvore DOM. 1. raiz, Tudo do javascript esta num objeto chamado window 2. dentro de window temos outros objetos ex * location (diz a localização do seu site/ url/ qual a pagina atual, qual a anterior)/ document( ) / history (de onde vc veio, par onde vc vai) dentro de document temos o child html, nesse caso document é parent de html dentro do html temos os childs head e body. Nesse caso html é parent de head e body é uma árvore hierárquica window > location / document / history document> html html> head / body head> meta / title body> h1 / p / p/ p/ div/ p/> strong
